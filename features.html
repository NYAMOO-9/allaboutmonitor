Features

1. 디스플레이 기술

모니터에 사용되는 주요 디스플레이 기술은 크게 LCD, LED, OLED, QLED 등으로 나눌 수 있습니다.

  • 장점: 저렴하고, 밝기와 해상도가 우수
  • 단점: 검은색 표현이 부족하고, 시야각이 좁을 수 있음

2. 해상도 (Resolution)

모니터의 해상도는 화면의 픽셀 수를 나타냅니다. 해상도가 높을수록 선명한 화면을 제공합니다.

  • HD (720p): 1280x720 pixel
  • Full HD (1080p): 1920x1080 pixel
  • Quad HD (QHD, 1440p): 2560x1440 pixel
  • Ultra HD (UHD): 3840x2160 pixel

3. 주사율 (Refresh Rate)

화면이 1초에 갱신되는 횟수를 나타낸다. 높은 주사율은 더 부드러운 화면 전환에 도움을 준다.

  • 60hz: 가장 보편적이고 일반적으로 사용된다
  • 120hz, 144hz, 240hz 이상: 주로 게임 및 고급 작업에 적합하다

4. 응답 속도(Response Time)

픽셀이 색을 바꾸는데 걸리는 시간으로 낮은 응답 속도를 가질수록 빠른 화면 전환이 가능해진다.

  • 1ms 이하: 게임과 같은 빠른 화면 전환이 이루어지는 환경에 적합
  • 5ms 이상: 보편적인 응답 속도

5. 시야각 (Viewing Angle)

화면을 정면이 아닌 각도에서 봤을 때 화면의 선명도를 말함.

  • 장점: 저렴하고, 밝기와 해상도가 우수
  • 단점: 검은색 표현이 부족하고, 시야각이 좁을 수 있음

6. 밝기 (Brightness)

화면의 밝기를 나타내며 nits라는 단위로 측정한다.

  • 400nits 이상: HDR 콘텐츠와 고급 디스플레이에 적합
  • 250~350 nits: 일반적인 사용 환경에 적합

7. 연결 포트

모니터에 사용되는 주요 입력 포트는 HDMI, DP, USB-C, VGA 등이 있습니다.

  • HDMI: 고화질의 영상/오디오 신호 전송에 적합
  • DP: 고해상도와 고주사율을 지원
  • USB-C: 데이터 전송 및 전력 공급
  • VGA: 현재는 사용하지 않는 포트
